Проект разделен на 2 паки:

* server - серверная часть 
* client - клиентская часть

Серверная часть - простой UDP-эхо сервер, который принимает пакеты и отправляет их обратно пользователю

Клиентская часть - простой пингер, который каждые 5 секунд отправляет пинг на сервер и принимает от него ответ

Как у клиента, так и у сервера свой Dockerfile и, соответственно, свой образ. Оба эти образа собираются в контейнеры через docker-compose файл
